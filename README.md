# AmazonScraping
scrap amazon website and extract new apple computers.

**extracted data:**
```
  -name of computer
  -number of like
  -price
  -image links
```


**run scraping with this command:**
```
      scrapy crawl amazon_spider
```

**for store that in json file:**
```
      scrapy crawl amazon_spider -o products.json
```
